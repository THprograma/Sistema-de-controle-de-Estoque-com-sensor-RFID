# Sistema de Monitoramento com Leitores UHF
Descrição

Este programa em Python faz a leitura de duas portas seriais (COM5 e COM6) conectadas a leitores UHF.
Ele identifica uma tag específica e registra na tela o horário e o local em que foi detectada.

A interface gráfica (Tkinter) mostra duas janelas: uma para a sala de ferramentas e outra para a sala dos tornos.

Funcionalidades

Leitura simultânea de duas portas seriais.

Identificação de uma tag alvo configurável.

Exibição de data, hora e local da detecção.

Controle de início e parada da leitura.

Intervalo mínimo entre exibições para evitar repetições.
